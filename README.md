こんなこと、ありませんか？

・発表、もうすぐなのに専門用語があやしい
・復習しようと思ってたのに、気づいたら前日

MAIORAL（マイオーラル）は、
そんな「発表前、時間ない！」を助けるアプリです。

やることはシンプル。
作る → 覚える → 確認する → 話せるようにする。

## 主な機能
・デッキ作成（手入力 / CSV / PDF / PPTX）
・フラッシュカード学習
・クイズで理解度チェック
・聞き流し（英訳→和訳、英訳のみ、ループ、再生間隔）
・通知でランダム単語リマインド
・発表向けリハーサル

つまり、
「覚えるだけ」で終わらず、
「本番で口から出る」まで持っていくのがMAIORALです。

Memorize + AI + Oral
発表前の追い込み、ちょっとだけラクにしませんか？
